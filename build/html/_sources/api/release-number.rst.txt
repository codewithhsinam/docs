Release Number Endpoint
==============

+-------------------------------------------------------------------------------+-------------------+-----------------+
| URL                                                                           | Required Values   | HTTP Methods    |
+===============================================================================+===================+=================+
| https://newtingtingapi.prixa.live/api/v1/phone-numbers/<phone_number_id>/     | Phone number id   | DEL             |
+-------------------------------------------------------------------------------+-------------------+-----------------+

Through the DELETE method of this endpoint, you will be able to release a number that you currently own.
